# 🧩 C++ Modules - Programmation Orientée Objet en C++

---

Le cursus CPP de 42 a pour objectif de découvrir progressivement le langage C++ à travers une série de modules. Chaque module introduit de nouveaux concepts de programmation orientée objet et de bonnes pratiques de développement moderne.

---

# ✅ Fonctionnalités principales

- Découverte de la syntaxe C++
- Création et utilisation de classes et d'objets
- Encapsulation et contrôle d'accès
- Héritage et polymorphisme
- Gestion de la mémoire dynamique
- Surcharge d'opérateurs
- Templates (modèles génériques)
- Gestion des exceptions
- Utilisation des conteneurs de la STL

---

# 🔧 Compilation

- `make` → Compile le programme
- `make clean` → Supprime les fichiers objets
- `make fclean` → Supprime les objets et l'exécutable
- `make re` → Recompile entièrement le projet

# 🏁 Objectif

Créer des programmes robustes et modulaires en appliquant les principes de la programmation orientée objet. Les modules CPP permettent de maîtriser progressivement le langage C++, la gestion de la mémoire, les structures de données et les bonnes pratiques du développement logiciel moderne.
